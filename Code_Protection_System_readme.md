Feature: Code Protection System (with @Symbol Integration)

Extends the CursorRIPER Framework with a comprehensive code protection system built on top of the @symbol enhancement. This integration prevents AI assistants from modifying critical code while leveraging symbol-based context awareness.

Key features:
- Multiple protection levels (PROTECTED, GUARDED, DEBUG, TEST, etc.)
- Symbol-aware protection comments that reference related @symbols
- Intelligent code scanning that leverages symbol relationships
- New @protection symbol type for referencing protected code in discussions
- Enhanced memory bank with combined symbol and protection registry
- Visual mapping of protection relationships within the symbol ecosystem
